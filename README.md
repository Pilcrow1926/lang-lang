# 난해한 프로그래밍 언어 "랑(Lang)" 문법 정리
이 언어는 해씨 수학으로부터 파생된 난해한 프로그래밍 언어인 "랑" 입니다.

## 1. 수 체계

### 기본 규칙
- **기본값**: 두(1), 근(3), 훌(5), 여(0)
- **대 접두사**: 한 숫자에 **한 번만** 사용 가능 (소수점 제외)
- **음수**: "느그" 접두사를 앞에 붙임
- **소수점**: "끔"으로 표현, 소수 부분은 각 자릿수를 독립적으로 표현

### 수 표현 알고리즘
어떤 수를 표현하려면:

1. **숫자 ÷ 6 = 몫 + 나머지**
2. **반복 횟수 = 몫 + 1** (기본값을 몇 번 반복할지)
3. **기본값 선택** (나머지에 따라):
   - 나머지 0 → 여
   - 나머지 1 → 두
   - 나머지 3 → 근
   - 나머지 5 → 훌
4. **대 접두사**: 나머지가 짝수(0, 2, 4)면 "대" 붙이기
5. **형식**: `[대] + [기본값] × 반복횟수 + 랑`

### 계산 예시

**100:**
- 100 ÷ 6 = 16 나머지 4
- 반복: 16 + 1 = 17번, 나머지 4 → 대근
- 표현: `대근근근근근근근근근근근근근근근근근랑`

**178:**
- 178 ÷ 6 = 29 나머지 4
- 반복: 29 + 1 = 30번, 나머지 4 → 대근
- 표현: `대근근근근근근근근근근근근근근근근근근근근근근근근근근근근근랑`

**1.5:**
- 정수 1: 1 ÷ 6 = 0 나머지 1 → 두랑
- 소수 5: 5 ÷ 6 = 0 나머지 5 → 훌랑
- 표현: `두끔훌랑`

```
뭐뭐두랑엇겸근랑엇  ((1) × 3)
```

## 5. 조건문과 반복문

### if/else 문법
```
엄 뭐 [조건식] 엇 호야 택
  [명령들]
틱
```

### else 문법
```
내꿈꿔 호야 택
  [명령들]
틱
```

### else if 문법
```
내꿈꿔 엄 뭐 [조건식] 엇 호야 택
  [명령들]
틱
```

### while 문법
```
장미 뭐 [조건식] 엇 호야 택
  [명령들]
틱
```

### for 문법
```
매우 [변수] 뒷 나가뭐[범위]엇 호야 택
  [명령들]
틱
```

**range 문법:**
- `나가뭐[끝]엇` : range(끝)
- `나가뭐[시작]그래[끝]엇` : range(시작, 끝)
- `나가뭐[시작]그래[끝]그래[스텝]엇` : range(시작, 끝, 스텝)

### 조건문 & 반복문 블록 기호
| 기호 | 의미 |
|---|---|
| 엄 | if / else if의 시작 |
| 뭐 | 조건/범위 열기 |
| 엇 | 조건/범위 닫기 |
| 호야 | : (colon) |
| 택 | 블록 시작 { |
| 틱 | 블록 끝 } |
| 내꿈꿔 | else / else if의 시작 |
| 장미 | while 반복문 시작 |
| 매우 | for 반복문 시작 (변수 선언) |
| 뒷 | in (for에서 사용) |
| 나가 | range |

## 8. 프로그램 예시


### 변수 선언 및 출력
```
매우 숫자 캬 근근랑
난뭐숫자엇
```

### 문자열 결합
```
매우 나이 캬 대훌훌훌랑
난뭐나이헐정살정엇
```
= print(나이 + "살")

### 조건문 (if/else)
```
매우 나이 캬 대훌훌훌랑
엄 뭐나이영대훌훌훌랑엇 호야 택
  난뭐정성인입니다정엇
틱
내꿈꿔 택
  난뭐정미성년자입니다정엇
틱
```
= 
```python
나이 = 18
if 나이 > 18:
    print("성인입니다")
else:
    print("미성년자입니다")
```

### 논리 연산자 포함
```
매우 나이 캬 대훌훌훌랑
엄 뭐나이영대훌훌훌랑심지어나이준느그대훌훌훌랑엇 호야 택
  난뭐정18세 이상 40세 미만입니다정엇
틱
```
= 
```python
나이 = 18
if 나이 > 18 and 나이 < 40:
    print("18세 이상 40세 미만입니다")
```

### while 반복문
```
매우 i 캬 여랑
장미 뭐i준대훌훌훌랑엇 호야 택
  난뭐i엇
  i허얼
틱
```
= 
```python
i = 0
while i < 18:
    print(i)
    i += 1
```

### for 반복문 (range 3개 인자)
```
매우 i 뒷 나가뭐여랑그래대훌훌훌랑엇 호야 택
  난뭐i엇
틱
```
= 
```python
for i in range(0, 18):
    print(i)
```

### for 반복문 (range 2개 인자)
```
매우 i 뒷 나가뭐대훌훌훌랑엇 호야 택
  난뭐i엇
틱
```
= 
```python
for i in range(18):
    print(i)
```

### for 반복문 (step 포함)
```
매우 i 뒷 나가뭐여랑그래대훌훌훌랑그래대두랑엇 호야 택
  난뭐i엇
틱
```
= 
```python
for i in range(0, 18, 2):
    print(i)
```
